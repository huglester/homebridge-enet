# eNet Homebridge plugin

The accessories added to eNet smart home will automatically appear in the Apple Home app if the below configuration is correct.

## Configuration

```

{
  "platform": "eNetPlatform",
  "username": "admin",
  "password": "admin",
  "host": "192.168.0.7"
}

```
